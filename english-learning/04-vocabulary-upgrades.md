# 📖 Vocabulary Upgrades – Simple to Formal

Use this table to upgrade your word choices in professional IT emails.

---

## General Formal Upgrades

| Simple Word | Formal Alternative |
|-------------|-------------------|
| checking / looking at | conducting a detailed analysis |
| fix | remediation / resolve |
| problem | anomaly / issue / concern |
| we know about | as you are aware |
| told us | confirmed |
| said | communicated / indicated |
| also | additionally / furthermore |
| but | however |
| so | therefore / consequently |
| about | regarding / pertaining to |
| for | in this regard / with reference to |
| followed up | actively following up to ensure timely resolution |
| need | require |
| see attached | please find attached |
| let us know | please feel free to reach out |
| delays | undue delays |
| more checks | enhanced validation |

---

## Status Update Words

| Basic | Professional |
|-------|-------------|
| looking good | all components are confirmed healthy |
| no problems | no anomalies detected |
| still checking | currently under active review |
| we fixed it | remediation has been completed |
| waiting | pending resolution |
| tracking it | this activity is being closely tracked |

---

## Email Opening and Closing Phrases

### Openings
- `With reference to your communication dated [date]...`
- `As discussed in our previous interaction...`
- `Further to the below, we would like to provide an update...`
- `In continuation of the above matter...`

### Closings
- `Should you require any further clarification, please feel free to reach out.`
- `We remain available for any further discussion or walkthrough.`
- `Thank you for your understanding and continued support.`
- `We will continue to proactively monitor and keep you informed.`

---

## IT-Specific Formal Phrases

| Situation | Phrase to Use |
|-----------|---------------|
| Raising a ticket | `A support case has been raised with [vendor] in this regard.` |
| Confirming health | `All components have been confirmed to be in a healthy and operational state.` |
| Noting an observation | `The following observations were identified during the review.` |
| Giving approval to proceed | `Please confirm your approval to proceed with the scheduled activity.` |
| Saying no issues | `No crash loops, reconciliation failures, or error events were observed during validation.` |
| Referring to Red Hat | `Engagement with Red Hat via the relevant open support case(s) will be initiated as required.` |
